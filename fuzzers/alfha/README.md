# SQLite3 Fuzzer Version 2 (alfha)

## 개요

`alfha`는 명세 기반 추가 최적화가 적용된 고급 SQLite3 퍼저입니다. `plain` (버전 1) 기반으로 SQLite3 명세서 분석을 통한 추가 개선사항이 적용되었습니다.

## 주요 개선사항 (vs plain)

### 1. 명세 기반 테스트 케이스 생성
- SQLite3 공식 문서 기반 SQL 구문 생성
- SQL-92/SQL-99 표준 준수 테스트
- 복잡한 쿼리 패턴 체계적 생성

### 2. 향상된 에러 시나리오
- 명세서의 에지 케이스 및 제약사항 테스트
- 타입 변환 경계값 테스트
- 메모리 및 리소스 한계 테스트

### 3. 고급 SQL 구문 지원
- 복잡한 조인 패턴
- 서브쿼리 및 CTE (Common Table Expression)
- 윈도우 함수 및 집계 함수 조합

## 빌드 및 실행

```bash
# 빌드
make alfha

# 실행
./alfha_standalone tests/testcases/sql/basic.sql
```
